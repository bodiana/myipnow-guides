# MyIPNow â€” Lightweight Network Diagnostic Tools

MyIPNow is a collection of simple, browser-based network utilities for quick diagnostics and verification.

The project focuses on clarity and speed: no accounts, no dashboards, and no unnecessary tracking.

---

## Purpose

When something breaks online, the cause is often unclear:

- Is a VPN active?
- Did the network change?
- Is DNS resolving correctly?
- Is the connection blocked or rate-limited?

MyIPNow provides fast answers to these questions using small, focused tools.

---

## Tools

### IP Address Lookup

The [IP address lookup](https://myipnow.net/) shows the public IPv4 and IPv6 address visible to websites and services.

Useful for:
- Verifying VPN or proxy usage
- Detecting network changes
- Understanding geo-based access restrictions
- Debugging security or authentication prompts

---

### DNS Lookup

The [DNS lookup tool](https://myipnow.net/dns-lookup) allows inspection of common DNS records:

- A / AAAA
- MX
- TXT

Typical use cases:
- Domain migrations
- CDN configuration checks
- Email setup verification
- Debugging partial site outages

---

### WHOIS Lookup

The [WHOIS lookup](https://myipnow.net/whois-lookup) provides domain and IP registration details such as:

- Registrar information
- Nameservers
- Registration dates

Useful for:
- Investigating unfamiliar domains
- Validating ownership
- Reviewing infrastructure changes

---

### Internet Speed Test

The [internet speed test](https://myipnow.net/internet-speed-test) measures connection performance, including:

- Download speed
- Upload speed
- Latency (ping)
- Stability (jitter)

Helps diagnose:
- Laggy connections
- Unstable video calls
- VPN or routing-related slowdowns

---

### IP Blacklist Check

The [IP blacklist checker](https://myipnow.net/ip-blacklist) checks whether an IP address appears on common reputation or blacklist databases.

Relevant for:
- Email delivery problems
- API access blocks
- Unexpected service denials

---

### Password Generator

The [password generator](https://myipnow.net/generate-password) creates strong, random passwords with configurable length and character sets.

Intended for:
- Generating unique credentials
- Avoiding predictable password patterns
- Improving basic security hygiene

---

## Typical Diagnostic Flow

1. Check the public IP using the [IP address lookup](https://myipnow.net/)
2. Measure performance with the [internet speed test](https://myipnow.net/internet-speed-test)
3. Verify domain records via the [DNS lookup](https://myipnow.net/dns-lookup)
4. Inspect ownership using the [WHOIS lookup](https://myipnow.net/whois-lookup)
5. If access fails, check reputation with the [IP blacklist checker](https://myipnow.net/ip-blacklist)

---

## Design Principles

- No accounts or logins
- No forced tracking
- Fast load times
- Mobile-friendly
- Focus on core functionality
